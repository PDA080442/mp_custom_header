Полный плавающий хедер Metaphysica (капсула + выпадающее меню), адаптив desktop/tablet/mobile, настройки в админке. Один шорткод.

== Использование ==

Вставьте в страницу / шаблон шапки Elementor / виджет «Короткий код»:

[mp_header]

Настройки — «MP Header» в левом меню админки: меню для колонок, телефоны, логотип, цвета, ширины по брейкпоинтам, «прилипание» сверху.

== Установка ==

1. Папку `mp-header` загрузите в `/wp-content/plugins/` (или ZIP через «Плагины → Добавить новый → Загрузить»).
2. Активируйте.
3. «MP Header» — настройте, сохраните.
4. Вставьте `[mp_header]`.

== Замечания ==

* Меню в колонках берутся из «Внешний вид → Меню» — сначала создайте нужные меню там.
* Родной хедер темы отключайте в настройках темы (иначе будет два хедера).
* Все стили и JS — внутри плагина, ничего не кладётся в «Дополнительный CSS».
